# 系统B · 战斗(战斗手册)

> 本页来源:Akun 的《战斗手册》(v2)。原始 Word 归档于 `design/_source/战斗手册-原始稿.docx`。
>
> 战斗全自动、回合制。怪兽是一组各自独立、各有 HP 的部件;攻击、闪避、格挡都**按部件**结算。

## 词条释义

| 词条 | 释义 |
|---|---|
| 部件 | 指单个的零部件，即某一个头、手、腿、躯干，以与插件相区分 |
| 部位 | 指一个类型位置的所有部件，如攻击部位：头就是攻击所有现存的头，攻击部位：手就是攻击所有现存的手 |
| 插件 | 可以安装在部件上的增强属性道具，如无特殊说明，每个部件只能安装1个插件 |
| 尾巴和插槽 | 尾巴和插槽是一种特殊的插件，均只能安装在躯干上且不占用躯干的插件位置，其中尾巴只能安装1条，插槽可以任意安装 |
| 伤害的属性 | 任意头 HP=0 → 该方下回合不行动 |
| 先攻顺序 | 伤害分为物理、火焰、冰冻、毒素四种类型，无特殊说明的伤害均为物理伤害 |
| 撕裂 | 造成持续的物理伤害 |
| 灼烧 | 造成持续的火焰伤害 |
| 冰冻 | 使1个部件停止行动1回合 |
| 中毒 | 造成持续伤害并降低攻击力 |
| 眩晕 | 怪兽下个回合停止行动 |

## 先攻

每回合开始前计算先攻,**根据先攻数值掷骰子**决定第一次攻击由谁发起,然后各部位轮流攻击。

- **公式(2026-07-03 拍板)**:先攻**每回合重掷**,A 先手概率 = `A先攻 ÷(A先攻 + B先攻)`;**一方先攻 > 0 而另一方为 0 时,先攻为正的一方固定先手**。
- 掷骰获胜方的 1 号腿先攻击,之后另一方的 1 号腿攻击,以此类推。

## 各部位攻击顺序

按 **腿 → 手 → 头** 的顺序进行。每类部位的每个部件分别编号(头1、头2;手1、手2;腿1、腿2……)。

## 指挥点(2026-07-07 拍板,Q12 方案A)

躯干与头提供**指挥点**,决定每回合有多少只手/腿能出手:

- **供给**:躯干自带基础指挥点 + 每个**存活的头**追加(具体数值见[零件表](01-assembly.md);单件数值仍可调)。
- **消耗**:每回合开始结算点数池;轮到某只**手/腿**攻击时消耗 1 点,**池空则该肢体当回合不攻击**(仍可被攻击、仍参与闪避/格挡)。消耗按攻击顺序自然进行(腿先、手后,同类内按槽位编号)。
- **头不消耗指挥点**(头是指挥者,自行行动)。
- **头被击破 → 下回合起指挥点立即减少**,肢体可能因此瘫痪——保护头有了实际价值。
- 指挥点每回合重置,不累积;死亡的部件不占点。

!!! note "数据背景"
    此机制来自[指挥度 A/B 模拟](sim-reports/2026-07-04-指挥度AB.md):无头流胜率 95.6%→30%,普通配装(1 头 + 4 肢)感觉不到约束。**躯干基础指挥是敏感参数**(调到 4 无头流就回到 80% 统治),改数值请重跑模拟器。
    2026-07-15 Akun 把躯干基础定为 **2/3/3**,[随机配装验证](sim-reports/2026-07-15-新零件表验证.md)通过:无头流从 40 强除名,前十全员带头 ✅

## 攻击的闪避

每次攻击发起后,根据角色的**腿1、腿2**提供的闪避计算总闪避,并提供给**全部部件**。攻击时先判定是否成功闪避。

- 例:腿1、腿2 各提供 5% 闪避 → 角色拥有 **10% 闪避**。
- 对应的腿**血量降为 0 后,其提供的闪避同时失效**。

## 攻击的优先级(选择目标)

| 攻击部位 | 目标规则 |
|---|---|
| **腿** | 优先攻击腿;无可用腿后,改攻击躯干 |
| **手** | 优先攻击手;无可用手后,改攻击躯干 |
| **头** | 每次 **50%** 攻击头、**50%** 攻击躯干 |

## 部位攻击的结算

每个部件按顺序发起攻击,**随机**命中对应部位的某个**存活**目标(2026-07-03 拍板:攻击次数不等时,多出的部件同样各自独立随机挑存活目标)。

- **暴击(2026-07-07 拍板,Q10)**:带暴击率的部件(目前是部分头,见零件表「暴击率(本体)」列)每次攻击独立判定,暴击时伤害 **×2(倍率待 Akun 确认,暂按 2 倍)**。暴击只对该部件**自己的攻击**生效。
- **多段攻击**:标注「双击」等的部件一回合攻击多次,**每段独立结算**目标/闪避/格挡(待 Akun 确认;目前仅 PVE 敌人猛犸象头使用)。

- 例(双方各 2 手):编号 手A1 / 手A2 / 手B1 / 手B2,攻击顺序与目标为:
  - A1 随机攻击 B1 / B2
  - B1 随机攻击 A1 / A2
  - A2 随机攻击 B1 / B2
  - B2 随机攻击 A1 / A2
- **一方攻击次数多于另一方**时,多出的部分在当前部位各自攻击结束后**集中攻击**。例 A1/A2 vs B1/B2/B3/B4:
  - A1 攻击 B1/B2/B3/B4,B1 攻击 A1/A2
  - A2 攻击 B1/B2/B3/B4,B2 攻击 A1/A2
  - B3 攻击 A1/A2,B4 攻击 A1/A2

## 头部的击破

一旦一方有**任意一个头部 HP 降为 0**,则其**下个回合不行动**;同一回合多个头部被击破,同样只是下个回合不行动。

- **「不行动」的范围(2026-07-03 拍板)**:所有部件**不主动攻击**,但**仍可被攻击、仍可闪避、仍可格挡**;眩晕只持续一个己方回合。
- 头被击破还会**永久失去该头提供的指挥点**(见上文「指挥点」),眩晕结束后肢体行动力也可能下降。

## 手部的格挡

当角色仍有手存在、且**头或躯干**受到攻击时,进行一次格挡判定:

- 基础格挡概率 **20%**。
- 格挡成功后,敌方造成的伤害 **×80%**,并**改由排序最后的存活的手**承受(2026-07-03 拍板:末位手已死则顺延到编号第二大的存活手,以此类推)。

## 胜利判定

**当一方躯干 HP 降为 0,即为战败。**(躯干是唯一的致命部位。)

## 可作为单元测试的规则点

!!! note "以下取自《战斗手册》,数值/逻辑明确,适合写成断言"

| 规则 | 用例 |
|---|---|
| 闪避加总 | 腿1 + 腿2 各 5% → 总闪避 10% |
| 闪避失效 | 提供闪避的腿 HP=0 → 该 5% 立即失效 |
| 格挡 | 头/躯干被击 + 有手 → 20% 触发;命中后伤害 ×80% 转最后一只手 |
| 头攻目标 | 头攻击:50% 命中头、50% 命中躯干 |
| 头被击破 | 任意头 HP=0 → 该方下回合不行动 |
| 先攻顺序 | 先攻 A>B → A.腿1 先手,B.腿1 次之,交替推进 |
| 胜负 | 躯干 HP=0 → 战败 |
