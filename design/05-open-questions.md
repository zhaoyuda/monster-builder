### 为什么是地下城冒险

炉石那个模式的循环是:**轻装开局 → 打一个 Boss → 三选一奖励 → 越打越强、Boss 越来越难 → 8 关通关或死了重开**。映射到我们:

| 炉石地下城冒险 | 手搓大怪兽 |
|---|---|
| 卡组 | 怪兽配装(本来就是一副 deck) |
| 三选一「主题卡包」 | 三选一「部件包」(攻击系 / 肉盾系 / 机制件系……) |
| 被动宝物(每隔几关给一次) | **你刚设计的 11 种插件**——等价物都不用发明 |
| 8 个 Boss 递增难度 | 6-8 场,按关卡序号给生物质预算(如 600→900→1300→1800→2400) |
| 死亡=重开一局 | run 归零重开,重开免费,换一条随机成长路线 |

三个对我们特别值钱的点:

1. **回答"随机还是人工"——折中,正是你猜的方向**:Boss 池**人工设计**(每只 Boss 一个机制个性,你的喷火头/芽孢手/碎骨锥正好一 Boss 一机制,原 L4-6 提案无缝并入),**出场顺序和奖励路径随机**。重玩性来自组合(Boss 顺序 × 三选一路线),不靠无限堆内容。
2. **随机三选一会逼玩家用没想过的配装**(这局没抽到好头?试试无头流)——怪奇造型从"自选彩蛋"变成"系统推着你体验",这正是"一个周末解出最优配装就弃坑"的解药。
3. **战力评估模拟器已经存在**:`sim/meta.py` 就是"给定生物质预算 → 随机采样上千合法配装 → 锦标赛评估强度"的工具([最近一次报告](sim-reports/2026-07-15-新零件表验证.md))。每只 Boss 进池前先跑几千局标定真实胜率,再人工调机制——这是我们比别的小团队多出来的能力,不用另做新工具。

### 对你几个具体问题的回答

- **PVE 流程是否合理**:合理。建议结构 = 现有 3 关(将来 3-5 关)教学序章保持线性,通关解锁「冒险模式」作为可重玩主体。
- **生物质上限放哪**:冒险模式下不需要全局上限——每个 run 内按关卡序号给预算(终局 ~2400,和模拟器验证过的量级衔接);教学关经济照旧。
- **体力/行动力系统**:**建议 MVP 明确不做,记入远期**。现在的目标是"让第三个人觉得好玩",单机 demo 加体力只会劝退试玩的朋友;它是商业化阶段的留存工具,不是好玩工具。
- **施工顺序(有硬依赖)**:三选一要有趣,靠的是选项之间有质的差异(机制件 vs 数值件 vs 插件)——纯数值件三选一是"选大的"伪选择。所以必须先做**机制引擎**(Q17 规则已齐):灼烧/芽孢/部件级眩晕/插件挂载 → 机制件进 demo → 冒险模式框架(Boss 池 + 三选一 + run 状态)。

### 补充三(2026-07-22):随机货架——把"每局不一样"补进 run(Yuda 提案,待拍板)

- **出发点**(Yuda 和团队讨论的观察):麻将、自走棋这类"百玩不腻"的游戏,共同引擎是**输入随机 × 多样流派**——摸牌/商店进货随机(逼你适应),多种胡牌方式/流派(保证怎么摸都有活路),而结算本身确定(输赢能归因)。我们的战斗层"结算确定"已经做得很好(镜像 50%±5% 验收),**缺的是输入随机**:现在 run 奖励固定生物质+全目录自选,是一道"解一次就毕业"的优化题,每个玩家最终都会收敛到同一套配装(模拟数据佐证:锦标赛前十 90% 是同一形态的微调变体)。
- **提案:run 内商店改"随机进货"**。你拍的经济骨架**完全不动**——奖励还是生物质、还是买件自己构筑;只改一件事:**每关战后,商店从已解锁零件里随机上架一部分**(建议 6-8 件,躯干常驻不进随机池)。这局没进猛爪?试试腿流/多头流——怪奇造型从"自选彩蛋"变成系统推着体验;两局 run 不再是同一道题。这正是自走棋商店的精髓,而且比三选一部件包实现还简单(就是个抽样过滤)。
- **可选加料**(可以后补,不影响主体):花少量生物质**刷新一次货架**(自走棋 reroll,给玩家一个"赌不赌"的决策);教学关完全不受影响,全目录照旧。
- **前置条件**(和 Q20 护栏三同理):先落地腿修法/手海治理——流派多样性到位之前上随机货架,随机到"没进手"的局等于判死刑,会变成挫败源。
